# Сайт-СМИ

Учебный проект - курсовая работа 3 курс РКЭ / сайт, перенесённый с InfinityFree на GitHub для демонстрации структуры и визуала.

## Что внутри
- HTML, CSS, JavaScript
- SQL-файл со структурой базы данных
- статическая версия сайта для просмотра

## Особенности
Проект изначально работал с PHP и MySQL на InfinityFree.
В репозитории сохранён визуал сайта и структура проекта для портфолио.

## База данных
SQL-файл находится в папке database/.

## Просмотр
Открой файл index.html или GitHub Pages.
